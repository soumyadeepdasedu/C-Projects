# 02 Simple Calculator

A C program that provides a menu-driven terminal interface to perform fundamental arithmetic operations.
This program runs in loop and does not end after every operator

---

## 🚀 Features

* **Addition** (`+`)
* **Subtraction** (`-`)
* **Multiplication** (`*`)
* **Division** (`/`)
* **Modulo** (`%`)
* **Power** (`^`)
